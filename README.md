# Vue Yellow Marker
Adds a text selection and highlighting functionality to your VUE components.

## Installation

```
npm install @kognity/vue-yellow-marker --save
```

## Usage

```javascript
import { HighlightMixin } from "@kognity/vue-yellow-marker";

export default {
  name: "MyComponent",
  mixins: [
    HighlightMixin,
  ],
  ...
  ...
}
```
